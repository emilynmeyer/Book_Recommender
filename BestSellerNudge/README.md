## Prerequisites 
Python 3.11.4

## Create and Activate Python Virtual Environment 

Windows:
- python -m venv venv
- env\Scripts\activate

Macos/Linux:
- python -m venv venv
- source venv/bin/activate

## Install Dependencies
pip install -r requirements.txt


## 1. Update Bestsellers List 
python update_bestsellers.py


## 2. Run Web App 
python app.py
